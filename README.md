# ⚾ 숫자 야구 게임
숫자 야구 게임 : 컴퓨터가 생성한 세 자릿수를 플레이어가 맞추는 게임

- 세 자릿수를 입력하되 숫자 중복을 허용하지 않음
- 1 ~ 9 사이의 숫자만 허용
- 스트라이크 : 숫자와 위치가 맞을 경우
- 볼 : 숫자는 맞지만 위치가 다를 경우
- 아웃 : 숫자와 위치 모두 틀릴 경우

# 1️⃣ Lv 1. 숫자 야구 게임 기본 구현

### 📋 요구사항
- 정답 숫자 생성하기
- 정답을 맞추기 위해 숫자를 입력하기
- 결과값 출력 및 게임 로직 적용하기
- 게임 이어서하기

### 💡 구현 전 정리
1. 컴퓨터는 1 ~ 9 사이의 숫자를 사용해 서로 다른 세 자릿수를 생성
2. 플레이어가 세 자릿수를 입력하고, 문자를 입력하면 예외 처리를 통해 오류 문구 출력
3. 플레이어의 입력값이 세 자릿수이면서 중복 값이 없는지 검증
4. 정답과 입력값을 비교하여 볼, 스트라이크, 아웃 중 무엇인지 출력
5. 3 스트라이크일 경우, 게임 종료
