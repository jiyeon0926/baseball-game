# ⚾ 숫자 야구 게임
숫자 야구 게임 : 컴퓨터가 생성한 세 자릿수를 플레이어가 맞추는 게임

- 숫자 중복을 허용하지 않음
- 1 ~ 9 사이의 숫자만 허용
- 스트라이크 : 숫자와 위치가 맞을 경우
- 볼 : 숫자는 맞지만 위치가 다를 경우
- 아웃 : 숫자와 위치 모두 틀릴 경우
